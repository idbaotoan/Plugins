How to install Demo Data

1. Download Jobsearch Plugin.

2. Install jobsearch-plugin.zip Under Plugin > Add New plugin

3. Activate the plugin Jobsearch 

4. After Activation the Jobsearch  Plugin then go to Appearance > Import Demo Data

5. Follow instruction in Import Demo Data all done.
